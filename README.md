## Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
```bash
git clone https://github.com/bakery-street-project/Polymorphic-Research-Framework.cd Polymorphic-Research-Framework
cd Polymorphic-Research-Framework
npm install
# or: yarn install
```

### Usage
```bash
# Start development
npm start

# Run tests
npm test

# Build for production
npm run build
```
